Data scientists and researchers can launch pre-configured RStudio Server instances to perform heavy statistical processing and data visualization.

## Launching RStudio

1. Select **RStudio Server** from the left navigation panel.
2. **Configure**: Choose your required CPU, Memory, and wall-time.
3. **Launch**: Click **Start RStudio**.

### Connecting with Access Token
Just like VS Code, RStudio requires an **Access Token** for secure browser access:

1. Click **Connect** on the active session.
2. Copy the **Access Token** provided in the MONTERA interface.
3. On the RStudio login page, enter your Montage **username** and paste the **Access Token** as the password.
