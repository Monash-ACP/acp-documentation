Some useful commands:

- `show_cluster` : This command lets you see available and occupied resources on the ACP!
- `show_job`: This command shows all your running/pending/cancelled jobs and most importantly VRAM usage for GPU users. Please utilize the maximum VRAM for your selected GPU!
- `chk_quota`: Show your directory usage and shared storage usage.
