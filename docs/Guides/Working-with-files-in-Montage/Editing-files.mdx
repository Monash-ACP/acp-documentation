## Through a text editor like VS Code (Recommended)

The most powerful way to edit files on the ACP is using **VS Code** with the **Remote - SSH** extension. This allows you to open folders and edit files directly on the server as if they were on your local machine.

1.  **Install VS Code**: Download it from [code.visualstudio.com](https://code.visualstudio.com/).
2.  **Install Extension**: Open VS Code, go to the Extensions view (`Ctrl+Shift+X`), and search for **"Remote - SSH"** (by Microsoft). Install it.
3.  **Connect**:
    - Click the "Remote Explorer" icon in the sidebar or press `F1` and type `Remote-SSH: Connect to Host...`.
    - Select **Add New SSH Host**.
    - Enter: `ssh <your-username>@10.168.151.1`
    - Choose a configuration file (usually the default one in your home directory).
    - Click **Connect** in the notification that pops up and enter your password when prompted.
4.  **Open Folder**: Once connected, click **Open Folder** in the explorer and navigate to your project directory (e.g., `/home/<username>`).

:::tip
This method is much better than uploading/downloading files manually because it handles everything in the background and gives you full access to VS Code's features (IntelliSense, terminal, etc.) directly on the server.
:::

## Through the terminal while connected on the ACP

If you have to edit a text file or a script while being in the terminal, connected to the ACP, the easiest way would be to use `nano`. Just get the path of the file you want to edit, and then:

```bash
nano /path/to/script_to_edit.sh
```
