By default, we provide **500 GB** of storage for each user. This is typically sufficient for most research projects. If you find yourself nearing this limit, please review our [Run out of storage quota?](./Run-out-of-your-storage-quota.mdx) guide.

## 📁 Standardized Reference Datasets

To avoid data redundancy and promote collaborative research, we provide shared storage for common databases and field-specific reference datasets.

### Group-Based Access
Access to these shared directories is determined by your **Linux User Group**. For example, to access the School of Engineering data (`/ibm/gpfs/soe`), your user must be part of the `soe` group.

To check which groups you currently belong to, run the following command in your terminal:

```bash
groups
```

**Sample Output:**
```text
soit
```

If you believe you should have access to a specific school directory but are not in the group, please contact your school's technical lead or the HPC team.

If you are using datasets that are commonly used in your field, please check the shared directories below. If you have a dataset you'd like to share with others, [let us know](https://monash-acp.github.io/acp-documentation/Contact-Us)! We can make it a shared resource for you.
