---
title: Sharing files to non-Montage users
sidebar_position: 5
---

You can share large files or datasets directly with colleagues who do not have an account on Montage using the **Web File Browser**. This generates a secure, timed link that you can share via email or messaging apps.

:::danger MUMVPN Required
Shared links are only accessible within the Monash internal network. If the recipient is working remotely, they **MUST** be connected to the [MUMVPN](https://www.monash.edu.my/it-services/infrastructure-and-security/vpn) to access the files.
:::

## 🚀 How to share files

1. **Context Menu**: Right-click on the file or folder you wish to share and select **Share**.

<img
  src={require('@site/static/img/Guides/Working-with-files-in-Montage/filebrowser_share_1.png').default}
  alt="Share Context Menu"
  style={{ marginBottom: '20px', borderRadius: '8px', boxShadow: '0 4px 8px rgba(0,0,0,0.1)' }}
/>

2. **Configure**: Select the **Share Duration** (how long the link will remain active) and optionally set a password. Click **SHARE**.

<img
  src={require('@site/static/img/Guides/Working-with-files-in-Montage/filebrowser_share_2.png').default}
  alt="Share Configuration"
  style={{ marginBottom: '20px', borderRadius: '8px', boxShadow: '0 4px 8px rgba(0,0,0,0.1)' }}
/>

3. **Copy Link**: The system will generate a unique link. Click the **Copy** icon to copy it to your clipboard.

<img
  src={require('@site/static/img/Guides/Working-with-files-in-Montage/filebrowser_share_3.png').default}
  alt="Generated Share Link"
  style={{ borderRadius: '8px', boxShadow: '0 4px 8px rgba(0,0,0,0.1)' }}
/>

---

:::tip Best Practices
- **Set Durations**: Always set a reasonable duration (e.g., 7 days) to ensure data security after the transfer is complete.
- **Passwords**: For sensitive research data, we strongly recommend setting an optional password and sharing it with your colleague via a separate, secure channel.
:::
